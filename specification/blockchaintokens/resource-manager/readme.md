## Getting Started
To build the SDK for Blockchain, simply [Install AutoRest](https://aka.ms/autorest/install) and in this folder, run:

> `autorest`

To see additional help and options, run:

> `autorest --help`

## Configuration



### Basic Information
These are the global settings for the Blockchain Token Service API.

``` yaml
openapi-type: arm
tag: package-2019-07-19-preview
```


### Tag: package-2019-07-19-preview

These settings apply only when `--tag=package-2019-07-19-preview` is specified on the command line.

``` yaml $(tag) == 'package-2019-07-19-preview'
input-file:
- Microsoft.BlockchainTokens/preview/2019-07-19-preview/blockchainTokens.json
```

``` yaml $(swagger-to-sdk)
  - repo: azure-sdk-for-js
  - repo: azure-sdk-for-java
  - repo: azure-sdk-for-ruby
    after_scripts:
      - bundle install && rake arm:regen_all_profiles['azure_mgmt_blockchain']
```
