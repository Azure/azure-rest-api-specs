# Microsoft Genome Resource Manager

This directory contains the Resource Manager (ARM) APIs for Microsoft Genome service.

## API Versions

### Preview Versions

## Resources

### Account
- **Description**: A Genome Account resource for managing genomic data processing
- **Operations**: Create, Read, Update, Delete, List
- **Properties**: Basic account properties with provisioning state
