# Windows Image Builder

> see https://aka.ms/autorest

---

## Getting Started

To build the SDK for Windows Image Builder, simply [Install AutoRest](https://aka.ms/autorest/install) and in this folder, run:

> `autorest`

To see additional help and options, run:

> `autorest --help`

