# Microsoft.ResourceNotifications Examples

This folder contains example files for the Microsoft.ResourceNotifications API version 2025-11-11-preview.

## Available Examples

### Operations
- `Operations_List.json` - Lists all available operations for the Resource Notifications provider

### Resource Subscriptions
- `ResourceSubscriptions_Get.json` - Retrieves a specific resource subscription
- `ResourceSubscriptions_CreateOrUpdate.json` - Creates or updates a resource subscription
- `ResourceSubscriptions_Update.json` - Updates an existing resource subscription
- `ResourceSubscriptions_Delete.json` - Deletes a resource subscription
- `ResourceSubscriptions_List.json` - Lists all resource subscriptions in a subscription

### Resource Subscription Destinations
- `ResourceSubscriptionDestinations_Get.json` - Retrieves a specific destination
- `ResourceSubscriptionDestinations_CreateOrUpdate.json` - Creates or updates a destination
- `ResourceSubscriptionDestinations_Update.json` - Updates an existing destination
- `ResourceSubscriptionDestinations_Delete.json` - Deletes a destination
- `ResourceSubscriptionDestinations_List.json` - Lists all destinations for a resource subscription

- Request parameters and body (if applicable)
